# e-calendar

Here You can add your meetings according to date and can edit and delete your meeting

I created this app using npx create-react-app

1. Install necessary dependencies

```bash
npm install
```

2. Run the app

```bash
npm start
```

3. Build the app

```bash
npm run build
```
